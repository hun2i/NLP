# ch 02 이론

## Autoencoders
- 오토인코더는 압축과 해제를 반복하며 특징 추출을 자동으로 학습
- 인코더로부터 나온 중간 결과물 (z)는 입력 (x)에 대한 feature vector
- 신경망을 통과시키는 것: 입력공간(input space) -> 잠재공간(latent space)로의 맵핑
	- 고차원 공간 -> 저차원 공간

## Hidden vector
- 각 레이어(Hidden layer)의 결과물
	- 모두 feature vector라고 볼 수 있음
- 해석이 굉장히 어려움
	- 중간 결과를 보기가 어려움
